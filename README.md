# Zentry

## Package Setup
##### ⭐️ Tailwind CSS 
- `npm install tailwindcss @tailwindcss/vite`
##### ⭐️ React Icons
- `npm install react-icons --save`
##### ⭐️ GSAP
- `npm install gsap @gsap/react`
    ```js
    import { gsap } from "gsap";
    import { useGSAP } from "@gsap/react";
    ```
